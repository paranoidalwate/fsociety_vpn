# fsociety_vpn

> "Everything is in your hands now. 
Society is a lie. Control is an illusion."

**fsociety_vpn** — это минималистичный и мощный инструмент для нативного раздельного туннелирования (Split Tunneling) в Linux. Он использует **Network Namespaces** (`ip netns`) для изоляции конкретных приложений внутри зашифрованного VPN-туннеля на базе протокола **AmneziaWG**, оставляя основную систему нетронутой.

Забудьте о глобальном VPN, который ломает локальную маршрутизацию или замедляет загрузку пакетов. Запускайте только то, что должно быть скрыто.

---

## Технические особенности

* Трафик изолируется на уровне ядра Linux. Приложение внутри туннеля физически не видит основной сетевой интерфейс.
* Полная поддержка параметров обфускации от DPI (Jc, Jmin, S1, S2).
* Приложения имеют доступ только к фантомному интерфейсу (`f_soc0`) и защищенному DNS (по умолчанию 1.1.1.1).
* Умный перехват пользователя позволяет запускать приложения графического интерфейса в один клик.

## Зависимости

Инструмент разработан и протестирован на **Arch Linux**. Для работы ядра сети требуются официальные модули AmneziaWG:

```bash
yay -S amneziawg-dkms amneziawg-tools
```

## Развертывание

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/ВАШ_НИКНЕЙМ/fsociety-vpn.git
cd fsociety-vpn/src
```
2. **Запустите установщик:**
```bash
sudo ./install.sh
```
3. **Конфигурация:**
Разместите ваш рабочий конфигурационный файл сервера (от AmneziaWG) строго по пути:
`/etc/amnezia/amneziawg/fsociety.conf`
4. **Активация подсистемы:**
```bash
sudo systemctl enable --now fsociety.service
```

## Использование

Просто добавьте `fsociety` перед любой консольной или графической командой. Все аргументы будут проброшены внутрь изоляции.

### Примеры:
```bash
# Запуск мессенджеров через туннель
fsociety Telegram
fsociety discord

# Проверка IP-адреса внутри туннеля
fsociety curl ifconfig.me

# Изолированный профиль браузера (рекомендуется)
fsociety firefox -P VPN
```
